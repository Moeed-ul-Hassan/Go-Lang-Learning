# VScode Shortcuts

> Here are Shortcuts I learn't While learning and creating Project.

## For Bars Closing 

For side bar Closing : _**CTRL + B**_
For Terminal Bar Closing : _**CTRL + J**_
For Agent Bar Closing : _**CTRL + ALT + B**_
For Agent Manager : _**CTRL + E**_

## Add Multiple Cursor for Multiline Editing.

Use Shortcut : _**Alt + Right Click**_
